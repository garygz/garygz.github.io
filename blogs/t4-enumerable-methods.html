Ruby Enumerable class provides a lot of very useful methods out of the box

Enumerable#group_by method

11/24/2014

Ruby Enumerable class offers a lot of useful methods which allow to easily manipulate the contents of object lists. One of these methods is Enumerable#group_by. group_by allows to group the elements into a map of arrays with the specified group names. This is a sample of the return value {:group1=>[1,2,3], :group2=>[3,4,5]}. It is incredibly flexible as it takes a Ruby block as an argument which should return a group key for each passed element. This is a sample of the block which groups an array of integers such as [1,2,3,4,5,6,7,8,9] into two groups: odd and even:


    [1,2,3,4,5,6,7,8,9].group_by {|element| element%2==0 ? "even":"odd"}
    #<= {"odd"=>[1, 3, 5, 7, 9], "even"=>[2, 4, 6, 8]}

Grouping of elements is a typical task for a programmer and this methods makes life so much easier by providing a one line method to do what otherwise would take at least 4 - 5 lines of code to implement.
